通过修改Defender扫描行为、扫描强度等设置来尽量降低Defender防护强度

Defender白名单注册表键值

在新一些的版本中有Tamper Protection所以就不行了

直接向白名单路径下添加值即可

  • 目录及文件：reg query "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Defender\Exclusions\Paths"
  • 后缀：reg query HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Defender\Exclusions\Extensions
  • 进程：reg query HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Defender\Exclusions\Processes
